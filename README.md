# customflags_flutter

A Flutter package for rendering customizable flags.

> ⚠️ **Early development** — this package is under active development and the API is not yet stable. Not recommended for production use.

## Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  customflags_flutter: ^0.0.1
```

Then run:

```sh
flutter pub get
```

## Usage

```dart
import 'package:customflags_flutter/customflags_flutter.dart';
```

A runnable example will live under [`example/`](example/) once added.
